Blenco Theme Core Plugin

== Description ==

Core plugin for the Blenco theme by RadiusTheme. Provides custom post types, Elementor widgets, and theme functionality.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/dentcure-core` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Make sure the Blenco theme is active.
